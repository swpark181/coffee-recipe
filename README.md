# ☕ 커피 레시피 추천 앱

에스프레소와 푸어오버 추출 방식에 맞는 레시피를 추천하고, 오늘 마신 커피 평가를 기록해 내일 분쇄도와 레시피를 자동으로 조정해주는 웹 앱입니다.

## 기능

- **레시피 추천** — 배전도(라이트~다크) + 분쇄도 입력 시 에스프레소/푸어오버 레시피 자동 생성
- **드리퍼 지원** — V60, 칼리타 웨이브, 케멕스, 오리가미, 에어로프레스
- **추출 기록** — 분쇄도, 도즈, 수율, 맛 평가(산미·쓴맛·바디·단맛) 저장
- **다음날 조정 제안** — 이전 평가(너무 셔요 / 써요 / 연해요 / 진해요)를 반영한 분쇄도·시간 조정안
- **국내 브랜드 DB** — 프릳츠, 테라로사, 커피리브레, 모모스, 빈브라더스 등 12개 로스터리 + 블렌드 정보
- **데이터 저장** — 브라우저 LocalStorage 사용 (설치·서버 불필요)

## 사용법

`coffee_recipe.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다. 별도 설치나 서버가 필요 없습니다.

## GitHub Pages로 배포하기

1. 레포 Settings → Pages → Branch: `main` / `(root)` 선택 후 Save
2. `https://swpark181.github.io/coffee-recipe/coffee_recipe.html` 로 접속

## 파일 구조

```
coffee_recipe.html   # 메인 앱 (단일 파일, 모든 기능 포함)
README.md
```
